# A Social Media Platform Using Django <br>

## Functionalities:

- Uploading a post
- Liking a post
- Creating an user account
- Signin
- Logout
- Account settings
- Customized Feed
- User authentication
- Follow & Unfollow users
- Searching a user
- User Suggestions
  
<br>

## Project Setup:

Create the project:
```
django-admin startproject social_platform
```
```
django-admin startapp core

```
Run the app:

```
python manage.py runserver
```
